# Contributing a Sponsor Booth

Only submit branding, images, copy, and links you have permission to publish.

1. Open the simulator and select **Design Booth**.
2. Build and preview the booth, then select **Download JSON**.
3. [Fork the repository](https://github.com/ljacobsson/reinvent-simulator/fork).
4. Add the downloaded file to `./booths` and its filename to `booths/index.json`.
5. Commit the changes and create a pull request.

<small>Real corporations interested in supporting this project should also consider [sponsoring Lars on GitHub](https://github.com/sponsors/ljacobsson).</small>
